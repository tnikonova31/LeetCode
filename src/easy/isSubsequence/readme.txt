Учитывая две строки s и t, верните true если s является подпоследовательностью t,
 или false в противном случае .

Подпоследовательность строки — это новая строка ,
которая формируется из исходной строки путем удаления некоторых (может быть ни одного)
символов без нарушения относительного положения остальных символов.
(т. е. "ace"является подпоследовательностью while is not)."abcde" "aec"

Пример 1:
Ввод: s = «abc», t = «ahbgdc»
 Выход: true

Пример 2:
Ввод: s = «axc», t = «ahbgdc»
 Вывод: false


Ограничения:
0 <= s.length <= 100
0 <= t.length <= 104
s и tсостоят только из строчных английских букв.
